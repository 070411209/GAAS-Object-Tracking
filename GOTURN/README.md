# GOTURN

## Dependencies
1. caffe
2. opencv
3. rospy

## Pretrained model
1. Download the pretrained model: 
```sh
wget http://cs.stanford.edu/people/davheld/public/GOTURN/trained_model/tracker.caffemodel
```
2. Move to the right directory:
```sh
mv tracker.caffemodel /path/to/GOTURN/nets/
```
